# k8s-Manifests

📦 Namespace

Q: What is Namespace?</br>
A: Separates resources inside a cluster.

⚙️ ConfigMap

Q: What is ConfigMap?
A: Stores non-sensitive configuration data.

🔐 Secret

Q: What is Secret?
A: Stores sensitive data like passwords and keys.

🚀 Deployment

Q: What is Deployment?
A: Runs and manages application pods.

🌐 Service

Q: What is Service?
A: Connects and exposes pods.

🔗 Ingress

Q: What is Ingress?
A: Gives external access using URL routing.

📈 HPA

Q: What is HPA?
A: Automatically scales pods based on load.

💾 PV

Q: What is PersistentVolume?
A: Provides storage in Kubernetes cluster.

📥 PVC

Q: What is PersistentVolumeClaim?
A: Requests storage from PV.

👤 ServiceAccount

Q: What is ServiceAccount?
A: Identity for pods inside cluster.

🔐 Role

Q: What is Role?
A: Gives permissions inside a namespace.

🔗 RoleBinding

Q: What is RoleBinding?
A: Attaches Role to a user or pod.

🧪 Job

Q: What is Job?
A: Runs a task once and exits.

⏰ CronJob

Q: What is CronJob?
A: Runs tasks on a schedule.

🧭 DaemonSet

Q: What is DaemonSet?
A: Runs one pod on every node.

🗄️ StatefulSet

Q: What is StatefulSet?
A: Runs stateful apps like databases with stable identity.
